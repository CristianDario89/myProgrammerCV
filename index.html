JavaScript

"Me maravilla usar JavaScript porque es un lenguaje tan sencillo y refinado con muchos propósitos. Me forme en JS cuando comencé a crear sitios web y más aún cuando descubrí que facilita la interacción con los visitantes.

Me complace mucho JS para resolver problemas algorítmicos. Siento que es fácil de usar y brinda características atractivas.

Después de mi experiencia programando API REST con node y angular para el intercambio de información entre el front y el back, quería desarrollar aún más mi comprensión de JavaScript. Diseñé y desarrollé una aplicación de software para gestionar ventas y entregas para distintos negocios mientras aprendía más sobre gestión de acceso para clientes externos, excluir datos no deseados y lo llamado click flow integration.

Me atrevo a decir que tengo un gran conocimiento de Javascript fundamentado en mi experiencia personal. "

Angular

"Aprendí a construir aplicaciones dinámicas de una sola página (SPA) hace ya 3 años. Mi interfaz gráfica para 'Reparto de bidones agua' descrita en mi portafolio dan fé de mis siguientes habilidades en Angular:

  • Facilidad para crear contenido dinámico a trávez del data-binding. Ahora mis componentes o elementos pueden comunicarse muy simplemente.
  • Asignar un comportamiento especial a las directivas en elementos del DOM.
  • Consumir servicios para la comunicación con la estructura de back-end a través de HTTP implementando el enrutamiento junto con el patrón de diseño basado en la inyección de dependencias.
  • conocimientos muy practicos al usar un almacén de datos y, cuando se modifica ese almacén, recibir automáticamente sus cambios, sin que tengamos que programar a mano ese tránsito de la información.
  • Programar con clases en TypeScript, lenguaje de programación moderno y orientado a objetos que proporciona una mejor estructura para gestionar grandes proyectos web.
  • Comprender que concede un mejor rendimiento a mayor escala."

Node

"Más de 4 años llevo programando software en Node - JavaScript de lado de servidor que utiliza un modelo asíncrono y dirigido por eventos - fortaleciendo mis skills en academias online como Udemy o estudiando por mi cuenta. Tras entender el concepto de que Node.js permite decirle al computador que hacer via JS he desarrollado apps web rápidas en tiempo real via conexiones bidireccionales.

Ahora puedo aportar las siguientes habilidades para su empresa:

  • Manejar peticiones HTTP del browser al servidor via puerto.
  • Mi código permanece mantenible y modular.
  • llamar funciones del routing para tratar lo que el usuario pida via navegador.
  • Procesar objetos recibidos del navegador a través de middlewares.
  • Dividir mi app en partes, a delegar tareas.
  • Procesar datos del usuario insertados en la interfaz.
